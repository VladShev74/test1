Основные команды git

"git clone" - клонирует удаленный репозиторий на ваш комп

"git add ." - говорит гиту отслеживать все изменения в вашем локальном репозитории

"git commit -m 'message'" - гит сохранит текущее состояние локального файла

"git push" - отправляет все изменения на удаленный репозиторий, то есть отправляет на GitHub

"git status" - показывает статус на локальном репозитории

"git pull" - стягивает все обновления с удаленного репозитория

"git log" - показывает историю всех коммитов

"git branch" - показывает список всех локальных веток

"git checkout -b 'name' " - создает новую ветку с именем 'name' и сразу переходит в неё

"git checkout 'name' " - переходит в ветку с именем 'name'
